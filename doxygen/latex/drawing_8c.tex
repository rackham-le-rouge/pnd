You need to inititalise it -\/$>$ e.g : drawLoadingBar(0, -\/1, 0, yourScreenLenght, 0);\par
 And after you can use it with the previous screen Lenght e.g : drawLoadingBar(2, 25, 100, -\/1, enumRed); // with the previous screen lenght\par
 You can also modify the screen lenght e.g : drawLoadingBar(2, 25, 100, newScreenLenght, enumRouge); // progress bar drawed with the new screen lenght\par
